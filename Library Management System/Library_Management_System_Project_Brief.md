# 📚 Library Management System (Python OOP Project)

## 🎯 Project Goal

Build a **console-based Library Management System** using
**Object-Oriented Programming (OOP)** in Python.

The goal is to practice real-world OOP concepts while building a clean
and organized project as a team.

------------------------------------------------------------------------

## 🧠 OOP Concepts We'll Use

-   Classes & Objects
-   Constructors (`__init__`)
-   Attributes & Methods
-   Encapsulation
-   Inheritance (where appropriate)
-   Polymorphism (where appropriate)
-   File Handling (later)
-   Exception Handling (later)

------------------------------------------------------------------------

## 📂 Suggested Project Structure

``` text
LibraryManagement/
│
├── main.py
├── book.py
├── library.py
├── member.py
└── data/
```

------------------------------------------------------------------------

## 📖 Main Features

-   ➕ Add a Book
-   ❌ Remove a Book
-   🔍 Search a Book
-   📚 Show All Books
-   📥 Borrow a Book
-   📤 Return a Book
-   💾 Save Data (later)
-   📂 Load Data (later)

------------------------------------------------------------------------

## 🧩 Classes

### Book

Stores information about a book.

**Attributes** - book_id - title - author - is_available

**Methods** - display_info() - borrow() - return_book()

------------------------------------------------------------------------

### Library

Manages all books.

**Methods** - add_book() - remove_book() - search_book() -
show_books() - borrow_book() - return_book()

------------------------------------------------------------------------

### Member

Represents a library member.

**Attributes** - member_id - name - borrowed_books

------------------------------------------------------------------------
